# Pybricks for FLL - Curriculum Plan

## Path Overview

**Path ID**: pybricks-fll
**Difficulty**: Intermediate
**Duration**: 12-16 weeks
**Category**: Competition
**Target Audience**: Ages 13+, requires Python & MicroPython

## Description

Pybricks for FLL is specialized training for First LEGO League competition programming. This path teaches the Pybricks library specifically optimized for FLL, including precise motor control, advanced sensor integration, mission programming, and optimization strategies for competition success.

By completing this path, students can:
- Master Pybricks library for EV3
- Implement precise autonomous navigation
- Program complex multi-objective missions
- Optimize for both speed and reliability
- Achieve competition-ready autonomous programs

## Prerequisites

- Python & MicroPython (Python fundamentals and MicroPython basics)
- Robotics Foundations (understanding of mechanics)
- Competition experience (preferred but not required)

## Level Structure

### Level 1: Pybricks Essentials (Weeks 1-2)
Introduction to Pybricks framework and EV3 programming.

**Modules:**
- Pybricks Fundamentals (library overview, setup, first program)
- Motor Control (speed, power, stopping methods, synchronization)

**Assessment**: Motor control checkpoint

### Level 2: Sensor Programming (Weeks 3-4)
Advanced sensor integration for autonomous navigation.

**Modules:**
- Sensor Integration (reading, calibration, filtering)
- Line Following and Navigation (PID control, gyro-based turning)

**Assessment**: Line following accuracy challenge

**Project**: Build reliable line follower

### Level 3: Mission Programming (Weeks 5-7)
Building complete autonomous mission sequences.

**Modules:**
- Mission Architecture (structure, state machines, error handling)
- Advanced Techniques (precision movement, attachment control)

**Assessment**: Multi-mission programming challenge

**Project**: Complete FLL autonomous mission

### Level 4: Competition Optimization (Weeks 8-9)
Preparing for actual FLL competition.

**Modules:**
- Performance Tuning (speed, reliability, accuracy)
- Team Programming (collaboration, code organization, version control)

**Capstone**: Full competition-ready autonomous routine
