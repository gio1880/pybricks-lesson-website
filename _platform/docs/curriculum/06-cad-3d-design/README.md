# CAD & 3D Design - Curriculum Plan

## Path Overview

**Path ID**: cad-3d-design
**Difficulty**: Beginner to Intermediate
**Duration**: 8-10 weeks
**Category**: Design
**Target Audience**: Ages 12+, no special prerequisites

## Description

CAD & 3D Design teaches digital design tools for creating robot parts and visualizing assemblies. Students progress from simple tools like Tinkercad to professional platforms like Onshape. This path enables students to design custom robot components and prepare designs for 3D printing.

By completing this path, students can:
- Use multiple CAD software platforms
- Design 3D models of robot components
- Create digital assemblies
- Prepare designs for fabrication
- Export designs in various formats

## Prerequisites

None, but Robotics Foundations provides useful context.

## Level Structure

### Level 1: 3D Design Fundamentals (Weeks 1-2)
Basic CAD concepts and Tinkercad.

**Modules:**
- Introduction to CAD (concepts, tools, workflows)
- Tinkercad Essentials (interface, shapes, basic design)

**Assessment**: Design a simple part in Tinkercad

**Project**: Design a storage box with specific dimensions

### Level 2: BrickLink Studio (Weeks 3-4)
Designing with LEGO bricks digitally.

**Modules:**
- BrickLink Basics (interface, brick library, placement)
- Robot Building in BrickLink (chassis, attachments, assemblies)

**Assessment**: Complete digital LEGO build

**Project**: Design a complete robot in BrickLink

### Level 3: Onshape Professional Design (Weeks 5-7)
Advanced CAD for custom parts.

**Modules:**
- Onshape Fundamentals (sketching, 3D features, assemblies)
- Advanced Modeling (complex features, design for manufacturing)

**Assessment**: Design and constraint verification

**Project**: Design a custom robot part for fabrication

**Capstone**: Complete mechanical assembly

## Sample Topics

**Level 1:**
- CAD Basics and Workflows
- 3D Coordinate System
- Basic Shapes and Primitives
- Grouping and Alignment
- Exporting Models

**Level 2:**
- LEGO Brick Selection
- Structural Design with Bricks
- Building Mechanical Attachments
- Assembly Instructions
- Exporting Designs

**Level 3:**
- Sketching in 2D
- 3D Features (extrude, revolve, sweep)
- Assemblies and Mates
- Design for 3D Printing
- Parametric Design

## Key Concepts

- 3D coordinate systems
- Sketch constraints
- Feature-based modeling
- Assembly relationships
- Design for manufacturing
- Tolerance and precision
- Part families and variants

## Assessment Strategy

**Formative:**
- Design exercises
- Part verification
- Assembly checks
- Dimension accuracy

**Summative:**
- Unit design projects
- Full assembly design
- Design documentation
- Presentation of designs
