# Robotics Foundations - Curriculum Plan

## Path Overview

**Path ID**: robotics-foundations
**Difficulty**: Beginner
**Duration**: 6-8 weeks
**Category**: Engineering
**Target Audience**: Ages 10+, no prior robotics experience required

## Description

Robotics Foundations is the essential starting point for anyone interested in robotics. This path covers the fundamental concepts and components that make all robots work, from simple machines to complex autonomous systems. Students learn about robot anatomy, mechanical systems, sensors, and control logic through hands-on exploration and projects.

By completing this path, students gain:
- Understanding of what makes something a "robot"
- Knowledge of robot components and their functions
- Familiarity with basic mechanical concepts (motors, gears, wheels)
- Introduction to sensors and environmental awareness
- Understanding of how robots make decisions and respond to their environment

## Prerequisites

None - this is the introductory path for the platform.

## Suggested Level Breakdown

### Level 1: What is a Robot? (Weeks 1-2)
Introduction and robot anatomy. Students explore what defines a robot and learn the basic components of robotic systems.

**Modules:**
- Introduction to Robotics (what robots are, why we build them)
- Building with LEGO (getting familiar with construction basics)

**Assessment**: Checkpoint quiz on robot types and components

### Level 2: Mechanical Systems (Weeks 2-3)
How robots move and exert force. Students learn about motors, gears, wheels, and mechanical advantage.

**Modules:**
- Motors and Motion (types of motors, power, control)
- Gears and Wheels (transmission, gear ratios, movement efficiency)

**Assessment**: Design challenge - create an efficient gear train

**Project Idea**: Build a wheeled robot with different gear ratios and test speed/power trade-offs

### Level 3: Sensing and Input (Week 4)
How robots perceive their environment. Students work with various sensor types and explore how sensors enable intelligent behavior.

**Modules:**
- Types of Sensors (touch, color, distance, gyro, other sensors)
- Sensor Applications (how sensors are used in real robots)

**Assessment**: Sensor identification and matching to applications

**Project Idea**: Build a simple obstacle detection system using sensors

### Level 4: Control and Logic (Weeks 5-6)
How robots make decisions. Students learn fundamental concepts of robot control logic including conditionals, loops, and event handling.

**Modules:**
- Basic Control Concepts (if-then logic, loops, decision-making)
- Program Structure (organizing robot behavior, state management)

**Assessment**: Logic puzzle challenges and simple control sequences

**Capstone Project**: Design and build a robot that can respond to multiple sensor inputs (light, touch, distance)

## Key Learning Outcomes

By the end of this path, students can:

1. **Define and classify robots**
   - Identify characteristics that make something a robot
   - Classify robots by type and application
   - Explain how robots differ from mechanical devices

2. **Understand mechanical systems**
   - Explain how motors create motion
   - Calculate gear ratios and predict speed/power outcomes
   - Select appropriate wheels and gearing for different tasks

3. **Work with sensors**
   - Identify different sensor types by function
   - Explain what information each sensor type provides
   - Match sensors to appropriate applications

4. **Think logically about robot behavior**
   - Understand conditional logic (if-then statements)
   - Understand loops and repetition
   - Trace through simple control sequences

## Topics to Cover

### Mechanical Concepts
- Torque and rotational force
- Linear and rotational motion
- Speed vs. torque trade-offs
- Gear ratios and gear trains
- Wheel types and traction
- Mechanical advantage
- Friction and resistance

### Sensors
- Touch sensors (switches)
- Color sensors
- Ultrasonic distance sensors
- Gyroscopic sensors
- Temperature sensors
- Motor encoders

### Control Concepts
- Boolean logic (true/false)
- Conditional statements (if-then-else)
- Loops (repeat N times, repeat forever)
- Events and triggers
- State and state transitions
- Sensor-driven behavior

### Real-World Applications
- FLL (FIRST LEGO League) robots
- FTC (FIRST Tech Challenge) robots
- Industrial robots
- Service robots
- Autonomous vehicles

## Notes for Content Development

### Writing Tips
- Use clear explanations of mechanical concepts
- Include lots of hands-on activities
- Connect to real robots and competitions
- Build gradually from simple to complex
- Include both theory and practical application

### Pedagogical Approach
- **Active Learning**: Hands-on building and experimentation
- **Discovery**: Students explore concepts through play
- **Scaffolding**: Gradually increase complexity
- **Real-World Connection**: Link to actual LEGO robots and competitions
- **Inclusive Design**: Accommodate different learning styles and abilities

### Challenges to Address
- Making abstract concepts (forces, motion) concrete
- Managing different experience levels in same classroom
- Ensuring all students can access physical robots
- Providing sufficient challenge for advanced students

### Potential Extensions
- Advanced gearing calculations
- Friction and efficiency measurements
- Sensor calibration techniques
- Advanced state machine design

### Common Student Misconceptions
1. Confusing motor torque with speed
2. Not understanding gear ratio direction
3. Thinking sensors "know" rather than "measure"
4. Assuming more motors = more power
5. Not recognizing the role of mechanical advantage

### Teaching Strategies
- Use demonstrations with transparent assemblies
- Have students predict before experimenting
- Compare different gear configurations directly
- Use metric measurements consistently
- Encourage experimentation and iteration
